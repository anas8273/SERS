# تحليل ملف تصاميم الهوية البصرية - 11 صفحة

## التصاميم الرئيسية:

### صفحة 1: غلاف أبيض بسيط
- خلفية بيضاء نظيفة
- ترويسة: شعار الوزارة (نقاط خضراء/تيل) يسار + "وزارة التعليم" + "Ministry of Education"
- يمين: المملكة العربية السعودية / وزارة التعليم / الإدارة العامة للتعليم ب.... / مدرسة
- خط فاصل عمودي أخضر/تيل بين اليمين واليسار
- شريط سفلي تدرج أزرق-أخضر (من الزاوية اليسرى)

### صفحة 2: أبيض مع عنوان في مستطيل بإطار أخضر
- نفس الترويسة
- مستطيل بإطار أخضر مدور الزوايا تحت الترويسة (لعنوان التقرير)
- يسار الترويسة: العام الدراسي + الفصل الدراسي

### صفحة 3: أبيض مع خطوط فاصلة
- نفس الترويسة لكن مع خطوط أفقية فاصلة بين كل سطر من بيانات الجهة
- مستطيل عنوان بإطار أخضر مدور
- شريط سفلي تدرج أزرق-أخضر

### صفحة 4: ترويسة داكنة تدرج أزرق-أخضر
- ترويسة بخلفية داكنة (تدرج من أزرق داكن #1a3a4a إلى أخضر تيل #0d7377)
- شعار الوزارة أبيض + نص أبيض
- شريط عنوان أخضر تحت الترويسة (خلفية خضراء + نص أبيض)
- محتوى أبيض
- فوتر: "التنفيذ: أ/ ..." و "مديرة المدرسة: أ/ ..."
- شريط سفلي تدرج

### صفحة 5: نفس 4 بدون خطوط فاصلة

### صفحة 6: ترويسة داكنة مع حقول بنمط جدول
- نفس الترويسة الداكنة
- شريط عنوان أخضر
- حقول بنمط جدول: عنوان الحقل في خلفية داكنة (تيل) + قيمة في خلفية بيضاء
- حقول قصيرة (منفذة البرنامج، الوصف الوظيفي) في صف واحد
- حقول طويلة (أهداف البرنامج، وصف البرنامج، الشواهد) في صف كامل
- فوتر داكن مع توقيعات

### صفحة 7: نمط edu-forms.com (المرجع الأساسي)
- ترويسة بيضاء مع شعار الوزارة
- خط فاصل عمودي
- عنوان في مستطيل بإطار أخضر
- حقول بنمط underlined (خط تحتي)
- حقول قصيرة في 3 أعمدة (المادة، الفئة المستهدفة، وقت التنفيذ)
- حقول طويلة بعرض كامل (وصف البرنامج، أهداف البرنامج، خطوات وآلية استخدامه)
- أثر البرنامج مع علامات صح ✓
- التوصيات بنقاط
- فوتر: ترقيم صفحات + www.edu-forms.com

## الألوان الرئيسية:
- تيل داكن: #1a3a4a أو #0d4d5a (ترويسة داكنة)
- تيل/أخضر: #0d7377 أو #0097A7 (خطوط فاصلة، إطارات)
- أخضر فاتح: #2d9e6e أو #3aaf85 (شريط عنوان)
- تدرج سفلي: من أزرق #1a5276 إلى أخضر #2ecc71

## عناصر مشتركة:
1. شعار وزارة التعليم (نقاط) دائماً موجود
2. خط فاصل عمودي بين الشعار وبيانات الجهة
3. مستطيل عنوان بإطار أخضر مدور الزوايا
4. شريط سفلي بتدرج أزرق-أخضر
5. التوقيعات: "التنفيذ" + "مديرة المدرسة" (ليس "مقيم")
6. حجم الخط كبير وواضح
7. المحتوى يملأ الصفحة
