# تحليل المتطلبات الجديدة vs الموجود

## 1. نظام تحليل النتائج
- **موجود:** صفحة analyses مع CSV input + إحصائيات بسيطة
- **مطلوب:** رفع Excel + رسوم بيانية تفاعلية (Charts) + تصنيف تلقائي (متفوقين/متعثرين) + تقرير PDF رسمي
- **الحل:** إعادة كتابة الصفحة بالكامل مع Charts + Excel parser + Auto-classification

## 2. الخطط العلاجية والإثرائية
- **موجود:** صفحة plans بسيطة
- **مطلوب:** مرتبطة بتحليل النتائج + AI يقترح أساليب + قالب رسمي
- **الحل:** إنشاء صفحة جديدة remedial-plans مرتبطة بالتحليل

## 3. الإنجاز والشواهد التلقائية
- **موجود:** صفحة achievements بسيطة + certificates
- **مطلوب:** مفكرة يومية + تقارير أسبوعية/شهرية تلقائية + مولد تقارير ذكي للأنشطة
- **الحل:** تحسين صفحة achievements بإضافة Daily Log + Auto-reports

## 4. التوليد الجماعي للشهادات
- **موجود:** صفحة batch-generate + certificates
- **مطلوب:** رفع Excel بأسماء 100 طالب + PDF واحد 100 صفحة + QR Code
- **الحل:** تحسين batch-generate لدعم هذا السيناريو

## 5. سجلات المتابعة والإدارة الصفية
- **موجود:** follow-up-log (API فقط)
- **مطلوب:** كشوف مناداة + كشف غياب + سجلات إدارية + تقييم سلوك + إشعار ولي أمر
- **الحل:** إنشاء صفحة classroom-management شاملة

## 6. المتجر والمحرك الديناميكي
- **موجود:** marketplace + editor + Form Builder + Template Mapper
- **مطلوب:** كل الخدمات تفتح كقوالب تفاعلية + إمكانية إضافة قوالب جديدة من لوحة الإدارة
- **الحل:** التأكد من أن المحرك الديناميكي يربط كل الخدمات بالقوالب
