# ملخص التعديلات المطلوبة على مشروع SERS

## 1. لوحة تحكم الإدارة (Super Admin Panel) - الأهم
### 1.1 محرك الأقسام الديناميكي (Dynamic Categories Engine)
- إضافة/تعديل/حذف أقسام وتصنيفات من الواجهة
- كل قسم له: اسم، أيقونة، حالة (نشط/مخفي)، نوع (مجاني/مدفوع/اشتراك)
- يظهر فوراً في المتجر والداشبورد

### 1.2 منشئ النماذج (Dynamic Form Builder) - Drag & Drop
- بناء الفورم لكل قالب من لوحة الإدارة
- أنواع الحقول: نص قصير، نص طويل، تاريخ، رفع ملف، قائمة منسدلة، رقم
- كل حقل مربوط بإطار على القالب (mapped_to)
- خيار تفعيل مساعد AI لكل حقل

### 1.3 محرر الإطارات (Template Mapper)
- رفع صورة التصميم (الخلفية)
- تحديد أماكن النصوص بصرياً (X, Y coordinates)
- تحديد نوع الخط واللون والحجم لكل إطار

### 1.4 إدارة الذكاء الاصطناعي (AI Prompt Manager)
- كتابة Hidden Prompt لكل قالب
- المتغيرات تُستبدل بمدخلات المستخدم
- تخصيص أسلوب الكتابة (رسمي، تحفيزي، تربوي)

## 2. المتجر الإلكتروني (Storefront)
### 2.1 نوعان من المنتجات
- **جاهز (Static)**: ملفات للتحميل المباشر (Word, PDF, PPT, Excel)
- **تفاعلي (Interactive)**: صفحة تعبئة بيانات → PDF جاهز

### 2.2 فلاتر وتصنيفات متقدمة
- تصفية حسب: القسم، النوع (جاهز/تفاعلي)، السعر، التقييم
- بحث ذكي
- نظام سلة وكوبونات ودفع

## 3. مساحة العمل التفاعلية (Interactive Workspace)
### 3.1 المحرر التفاعلي
- عرض الفورم الديناميكي حسب القالب المختار
- المعاينة الحية (Live Preview)
- زر مساعد AI لتعبئة الحقول
- تصدير PDF عالي الدقة

### 3.2 التوليد الجماعي (Bulk Generation)
- رفع Excel/CSV بالأسماء
- توليد مئات الشهادات/التقارير دفعة واحدة
- إضافة QR Code لكل ملف

## 4. الخدمات التعليمية (SaaS Tools)
### 4.1 تحليل النتائج (Results Analyzer)
- رفع Excel بالدرجات
- رسوم بيانية تفاعلية (Pie, Bar, Line)
- فرز ذكي (أعلى/أقل طلاب)
- زر إنشاء خطة علاجية/إثرائية
- زر إصدار شهادات جماعية للمتفوقين
- تقرير تحليلي بالذكاء الاصطناعي

### 4.2 الخطط التعليمية
- خطط علاجية (للطلاب الضعاف)
- خطط إثرائية (للمتفوقين)
- تصدير PDF رسمي

### 4.3 التقارير والشواهد
- إنجاز يومي/أسبوعي/شهري
- تقارير برامج ومبادرات
- شواهد أداء وظيفي
- AI يصيغ التقارير من نقاط مختصرة

### 4.4 الشهادات والبطاقات
- قوالب متنوعة (شكر، تفوق، مشاركة)
- توليد جماعي
- QR Code للتحقق

### 4.5 سجلات المتابعة
- كشوف المناداة
- المتابعة اليومية
- ملف المعلم الشامل

## 5. الذكاء الاصطناعي
- مساعد صياغة (لا يصمم، بل يكتب المحتوى)
- Hidden Prompts مخصصة لكل قالب
- توليد تقارير من نقاط مختصرة
- توليد خطط علاجية/إثرائية
- تحليل بيانات الدرجات

## 6. قاعدة البيانات الهجينة
### MySQL (Laravel)
- users, roles, plans, subscriptions
- categories, templates_catalog
- orders, payments, coupons

### Firebase Firestore
- Template_Canvas (إحداثيات الإطارات)
- Dynamic_Forms (حقول الإدخال)
- AI_Prompts (التعليمات المخفية)
- User_Records (مدخلات المستخدمين)
