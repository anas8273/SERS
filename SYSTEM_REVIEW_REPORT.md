## 4. هيكل الملفات الرئيسية

```
SERS/
├── frontend/                    # Next.js 15 + React 19
│   ├── src/
│   │   ├── app/

│   │   │   ├── (dashboard)/    # لوحة تحكم المستخدم
│   │   │   ├── marketplace/    # سوق القوالب
│   │   │   ├── services/       # الخدمات التعليمية
│   │   │   ├── editor/         # المحرر التفاعلي
│   │   │   └── ...
│   │   ├── components/         # المكونات
│   │   ├── lib/               # API و utilities
│   │   ├── i18n/              # الترجمات
│   │   └── stores/            # Zustand stores
│   └── ...
├── backend/                    # Laravel 11
│   ├── app/
│   │   ├── Http/
│   │   │   ├── Controllers/   # API Controllers
│   │   │   └── Middleware/    # الصلاحيات والأمان
│   │   ├── Models/            # Eloquent Models
│   │   └── Services/          # AI Service
│   ├── database/
│   │   └── migrations/        # جداول قاعدة البيانات
│   └── routes/
│       └── api.php            # API Routes
└── ...
```
