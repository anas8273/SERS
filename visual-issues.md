# مشاكل التصميم المكتشفة

## 1. الأيقونات المتراكبة / غير مرتبة
- الأيقونات emoji (📋, 🤝, 👨‍👩‍👧, 🎯, etc.) تظهر كمربعات رمادية صغيرة بدلاً من emoji ملونة
- بعض الأيقونات تظهر كمربعات فارغة (مثل أداء الواجبات الوظيفية)
- الأيقونات صغيرة جداً ومتراكبة مع النص
- كل بند يعرض: أيقونة emoji + اسم البند + عدد المؤشرات + الوزن + التقييم + عدد الشواهد + سهم
- المشكلة: كل هذه العناصر في صف واحد مما يسبب ازدحام

## 2. مشكلة التصنيف الذكي
- عند الضغط على "رفع شاهد مع تصنيف ذكي" يظهر file input مخفي
- عند رفع ملف، handleSmartUpload يحاول:
  1. رفع الملف لـ S3 عبر portfolio.uploadFile (يحتاج auth)
  2. إذا فشل الرفع لـ S3، يرسل base64 للـ AI
  3. لكن إذا حدث خطأ UNAUTHORIZED، main.tsx يعيد التوجيه لصفحة تسجيل الدخول

## الحل المطلوب
1. استبدال emoji بأيقونات Lucide React ملونة
2. تحسين تخطيط البطاقات لتكون أكثر وضوحاً
3. إصلاح handleSmartUpload ليتعامل مع حالة عدم تسجيل الدخول بشكل صحيح
