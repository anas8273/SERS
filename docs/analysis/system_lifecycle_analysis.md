## 5. تحليل دورة حياة النظام

### 5.1 سيناريو المستخدم الكامل

```
1. التسجيل/تسجيل الدخول
   ↓
2. تصفح القوالب والخدمات
   ↓
3. اختيار قالب (جاهز أو تفاعلي)
   ↓
4. إذا تفاعلي:
   - تعبئة البيانات مع اقتراحات AI
   - معاينة القالب
   - تصدير (PDF/صورة)
   ↓
5. إذا جاهز:
   - إضافة للسلة
   - الدفع
   - التحميل
   ↓
6. استخدام الخدمات التعليمية:
   - تحليل النتائج
   - إنشاء الشهادات
   - إعداد الخطط
   - توثيق الإنجازات
   - تقييم الأداء
   ↓
7. المساعد الذكي للدعم
```

### 5.2 سيناريو المدير

```
1. تسجيل الدخول كمدير
   ↓
2. لوحة التحكم الرئيسية
   - إحصائيات النظام
   - آخر الطلبات
   - المستخدمين الجدد
   ↓
3. إدارة المحتوى:
   - إضافة/تعديل/حذف القوالب
   - إدارة التصنيفات
   - إدارة المنتجات
   ↓
4. إدارة المستخدمين:
   - عرض المستخدمين
   - تعديل الصلاحيات
   ↓
5. إدارة الطلبات:
   - معالجة الطلبات
   - تتبع المدفوعات
   ↓
6. التقارير والإحصائيات
```
