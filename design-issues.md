# مشاكل التصميم المكتشفة

## 1. الأيقونات المتراكبة
عند النظر للبطاقات، كل بطاقة بند تحتوي على:
- أيقونة emoji في مربع (يسار)
- عنوان البند + عدد المؤشرات + الوزن (وسط)
- درجة 0/5 + عدد الشواهد + سهم (يمين)
- شريط تقدم أسفل

المشكلة: الأيقونة الأولى (📋) تظهر كمربع فارغ/clipboard icon وليس emoji ملون
المشكلة الأكبر: العناصر في كل بطاقة كثيرة ومتزاحمة - خصوصاً على الشاشات الصغيرة
الأيقونة والدرجة والعدد والسهم كلها في سطر واحد مع العنوان

## 2. التصنيف الذكي
عند رفع ملف:
- يتم فتح file picker
- بعد اختيار الملف، يتم إرسال base64 للـ AI
- إذا حدث خطأ في الـ mutation (مثل timeout أو خطأ شبكة)، الصفحة تختفي
- المشكلة: error handling ضعيف - الخطأ يسبب re-render أو redirect
