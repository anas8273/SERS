# مشاكل التصميم الحالية

## 1. الأيقونات
- الأيقونات الآن Lucide React ملونة وتعمل بشكل جيد ✓
- لكن الخلفية خفيفة جداً (opacity 15%) - تحتاج تكون أوضح

## 2. تخطيط البطاقات
- البطاقات حالياً في صف واحد (list) - جيد لكن يمكن تحسينه
- الإحصائيات (0/5, 0 شاهد) على اليسار - جيد
- شريط التقدم للمؤشرات موجود ✓

## 3. مشكلة التصنيف الذكي
- classifyEvidence هي publicProcedure ✓
- handleSmartUpload يرسل base64 مباشرة ✓
- المشكلة المحتملة: حجم base64 كبير جداً يسبب timeout
- أو: النتيجة لا تتطابق مع أي criterion.id

## 4. ما يحتاج إصلاح
- تحسين ألوان خلفية الأيقونات
- اختبار التصنيف الذكي فعلياً
- التأكد من أن النتيجة تتطابق مع criterion IDs
