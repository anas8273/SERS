## 1. صفحة القوالب (السوق) - خيار جاهز/تفاعلي

### الملف: `/frontend/src/app/marketplace/[slug]/page.tsx`

**التعديلات:**
- إضافة خيار اختيار نوع القالب (جاهز أو تفاعلي)
- **القالب الجاهز**: يضاف للسلة → الدفع → تحميل PDF مباشرة
- **القالب التفاعلي**: يضاف للسلة → الدفع → الانتقال لصفحة المحرر لتعبئة البيانات مع الذكاء الاصطناعي

**الميزات الجديدة:**
- واجهة اختيار واضحة مع أيقونات
- شرح مفصل لكل خيار
- badges توضح ميزات كل نوع (اقتراحات ذكية، تخصيص كامل، تحميل فوري، إلخ)

## 2. قسم الخدمات التعليمية

### الملفات:
- `/frontend/src/app/services/page.tsx` (صفحة جديدة عامة)
- `/frontend/src/app/services/[slug]/page.tsx` (صفحة تفاصيل الخدمة)

**التعديلات:**
- نقل صفحة الخدمات لتكون عامة (بدون حماية تسجيل الدخول)
- إضافة Navbar و Footer للصفحات

**الخدمات المتاحة:**
1. **تحليل النتائج** - تحليل نتائج الاختبارات واستخراج التقارير
2. **الشهادات والتقدير** - إنشاء شهادات الشكر والتقدير والتخرج
3. **الخطط التعليمية** - إنشاء الخطط العلاجية والإثرائية وتوزيع المنهج
4. **توثيق الإنجازات** - توثيق الإنجازات اليومية والأسبوعية والشهرية
5. **تقييم الأداء** - إدارة تقييمات الأداء الوظيفي وشواهد الأداء
6. **الاختبارات** - إنشاء وإدارة الاختبارات وتسجيل درجات الطلاب
7. **المساعد الذكي** - مساعد ذكي يساعدك في إعداد الخطط والتقارير
8. **قوالبي المحفوظة** - إدارة القوالب المحفوظة والمشتراة والمفضلة

**صفحة تفاصيل كل خدمة تحتوي على:**
- وصف مفصل للخدمة
- قائمة المميزات
- الفوائد الرئيسية
- خطوات كيفية الاستخدام
- إحصائيات الاستخدام
- خطط الأسعار
- الأسئلة الشائعة
- الخدمات ذات الصلة

## 5. دورة حياة النظام

### سيناريو القالب الجاهز:
1. المستخدم يتصفح السوق
2. يختار قالب
3. يختار "جاهز"
4. يضيف للسلة
5. يدفع
6. يحمل PDF مباشرة

### سيناريو القالب التفاعلي:
1. المستخدم يتصفح السوق
2. يختار قالب
3. يختار "تفاعلي"
4. يضيف للسلة
5. يدفع
6. ينتقل لصفحة المحرر
7. يعبئ البيانات مع مساعدة الذكاء الاصطناعي
8. يصدر PDF

### سيناريو الخدمات التعليمية:
1. المستخدم يتصفح الخدمات
2. يختار خدمة (مثل تحليل النتائج)
3. يرى التفاصيل والمميزات
4. يضغط "ابدأ الاستخدام"
5. ينتقل للصفحة المخصصة للخدمة
